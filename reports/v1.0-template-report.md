## 1. Overview

The v1.0-template release established the foundational static website for the Jenna Lynn Photography project. It was built as a fully responsive, self-contained photography portfolio and client-gallery template using vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, no external runtime dependencies. The site shipped under a generic placeholder brand ("Grace & Lens") so that the structure and design system could be validated before real client content was integrated in v2.0.

This version includes the main marketing site (`index.html`) plus a separate password-protected client gallery (`gallery.html`). All imagery in this version is placeholder content — gray boxes annotated with recommended image dimensions — so the template can be handed off and populated with real photography later.

**Total files in project:** 16

## 5. Features

### Navigation & Layout
- **Navbar scroll effect** — navbar becomes opaque/shadowed on scroll.
- **Mobile hamburger menu** — collapsible menu for small viewports.
- **Responsive design** — grid and layout adapt across desktop, tablet, and mobile breakpoints.
- **Scroll reveal animations** — sections animate into view on scroll.

### Portfolio
- **Portfolio filtering** — filter by All, Weddings, Couples, and Details.
- **Lightbox with keyboard navigation** — arrow keys, Escape to close.

### Testimonials
- **Testimonial slider with auto-rotate** — transitions between reviews automatically.

### Contact
- **Contact form with success message** — client-side success confirmation on submit.

### Client Gallery
- **Password-protected client gallery** (`gallery.html`) — passcode gate before viewing photos.
- **Slideshow** — automatic image cycling within a gallery.
- **Favorites** — clients can mark favorite photos.
- **Download** — clients can download images.
